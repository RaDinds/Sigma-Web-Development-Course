Important Core Web Vitals

1. CLS stands for cumulative layout shift

means "page ka content zada shift nhi hona chaiya"

2. LCP stands for Largest Contentful Paint

means "page ka sabsa bara element 2.5 sec ma load hona chaiya"

3. FID stands for First Input Delay

means "koi bhi input 100 milliseconds ma process start ho jana chaiya"
